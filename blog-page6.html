[#] Попробывал подкинуть TS в уже существуюший проект и столкнулся с ошибкой, VSCode ругался на знак @ в запросе:
"npm install --save typescript @types/node @types/react @types/react-dom @types/jest"
Решил эту проблему простым способом: перешел в папку с проектом и открыл папку с помощью Git Bash Here (терминал Git). И уже в нем выполнил инстал:
"npm install --save typescript @types/node @types/react @types/react-dom @types/jest"
Ну и после установки соответственно неперезагрузил проект и потерял минут 5 на поиски проблем. В дальнейшем буду всегда после инстала каких либо пакетов перезагружать проект.
