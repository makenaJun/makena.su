[#] Не знаю насколько это правильно, я просто добавил в ссылку для подключения JS файла, код для проверки даты изменения JSON файла. Ссылка на файл имеет вид:
http://makena.su/js/progress.js?16135431121613543292

При любом изменении файла JS или JSON ссылка будет менятся и сооветственно браузер будет обновлять кэш. Единственный минус, при изменении всего одного файла будут обновляться два, что повлечет за собой задержки при загрузке. В моем случае файлы маленькие, так что думаю пойдет. Для более нагруженных сайтов придется искать другой способ.

<script src="js/progress.js?<?php echo filemtime('js/progress.js'); echo filemtime('js/data-progress.json') ?>"></script>
